Articles contain a block of metadata information called "frontmatter" a the top of the file.

```mdx title="src/content/docs/example.mdx"
---
title: Example Article
description: A demonstration of how articles are structured
---

This is the content of the example article!
```

The only required attribute is `title`.

See the Starlight frontmatter reference for the full list of properties available:

https://starlight.astro.build/reference/frontmatter/
